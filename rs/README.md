# sqler (single project)

Unified Rust library providing database connections and export utilities with a C ABI for Qt.

Backends and features (default):
- SQL via `sqlx::Any` (Postgres/MySQL/SQLite)
- MongoDB
- Redis
- Export CSV（基于 SQLx 查询写出 CSV；支持 SQLite/Postgres/MySQL）
- Optional: Oracle (`with-oracle`), SQL Server placeholder (`with-mssql`)

## Build

- Default (includes export-sqlite): `cargo build --release`
- Enable Oracle: `cargo build --release --features with-oracle`
- Enable SQL Server placeholder: `cargo build --release --features with-mssql`
  （导出能力依赖 sqlx）

This crate builds `rlib` and `cdylib`. Load from Qt via `QLibrary` or link.

## C ABI

Header is generated by cbindgen at build time to `include/sqler.h`.
If you build outside the provided script, run:
`cbindgen --config cbindgen.toml --crate sqler --output include/sqler.h`.

Summary:
- Lifecycle: `sqler_new`, `sqler_free`, `sqler_version`, `sqler_last_error`, `sqler_string_free`
- Connections: `sqler_connect(ctx, kind, uri) -> id`, `sqler_disconnect(ctx, id)` (1=PG, 2=MySQL, 3=SQLite, 4=MSSQL, 5=Oracle, 6=MongoDB, 7=Redis)
- SQL: `sqler_run_sql(ctx, id, sql) -> char*` (JSON)
- Redis: `sqler_redis_set`, `sqler_redis_get` (JSON with base64)
- MongoDB: `sqler_mongo_insert_one`, `sqler_mongo_find_one`
- Export (SQLite via rusqlite): `sqler_export_init_logger`, `sqler_export_run_json`, `sqler_export_run`
- Export (sqlx helper): `sqler_export_query_to_csv(ctx, id, query, output_path, include_headers, delimiter)`

## JSON formats

- SQL SELECT: `{ "columns": [..], "rows": [[..], ..] }` (stringified cells)
- SQL non-SELECT: `{ "rows_affected": <u64> }`
- Redis get: `{ "data_base64": "..." }` or `null`
- Mongo insert: `_id` as JSON
- Mongo find_one: Document JSON or `null`

## Qt Usage (C++)

```
#include "sqler.h"

auto ctx = sqler_new();
uint64_t pg = sqler_connect(ctx, 1, "postgres://user:pass@localhost:5432/db");
char* res = sqler_run_sql(ctx, pg, "select 1 as one");
// parse JSON (QJsonDocument)
sqler_string_free(res);

// Export using existing connection (sqlx)
int rc = sqler_export_query_to_csv(ctx, pg, "select * from users", "/tmp/users.csv", 1, ',');

sqler_free(ctx);
```

Notes:
- Mongo URIs should include a default database (e.g. `mongodb://host:27017/mydb`).
- Redis uses blocking client offloaded to worker threads.
- Oracle SELECT and MSSQL are not implemented in FFI; Oracle non-SELECT minimal support exists.
- Always call `sqler_string_free` for returned strings (or `sqler_export_free_string`, alias).
