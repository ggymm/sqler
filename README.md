### 项目目标

跨平台数据库管理软件

#### 前提

1. 使用QT，C++，RUST，CMake构建项目（QT代码在src中，其他库代码在lib中）
2. 分离UI和业务代码，业务代码在lib中实现，使用rust编写，导出库给QT应用调用
3. 项目需要兼容亮色和暗色主题，所有页面使用的颜色，边距，长宽高等信息，都要在主题中统一配置，需要注意命名空间问题，不要冲突
4. 遇到重复的，复杂的代码需要考虑精简问题
5. 使用makefile编译和调试项目


#### 主体框架

1. 构建顶，左，右的布局，顶部是图标菜单，左侧是连接管理，右侧是主要内容区域
2. 添加顶部菜单，顶部菜单为：新建连接，新建查询，分割线，表，查询，函数，用户
3. 点击顶部菜单新建连接之后，弹出子窗口，现选择数据库类型再编辑数据库连接信息
4. 为每个数据库类型都创建独立的连接编辑页面
5. 资源统一放在了assets目录下


#### 新建连接窗口

1. 选择文件类型窗口，左侧是图标，右侧是数据库类型，一行一个，文本要对齐
2. 数据库连接编辑窗口，底部左侧要有测试连接，底部右侧要有上一步，取消，保存按钮
3. 数据库连接编辑窗口，输入框全部保持文本，并且充满整行，label要注意显示完整
