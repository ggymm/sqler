### 项目目标

完全参考 navicat 构建一个数据库管理软件

#### 任务

1. 构建项目结构，使用QT6，QML，C++20，CMake管理项目
2. 构建顶，左，右的布局，顶部是图标菜单，左侧是连接管理，右侧是主要内容区域
3. 项目需要兼容亮色和暗色主题，所有页面使用的颜色，边距，长宽高等信息，都要在主题中统一配置，需要注意命名空间问题，不要冲突
4. 遇到重复的，复杂的代码需要考虑精简问题


1. 添加数据保存模块，用于保存连接信息
2. 顶部菜单为：新建连接，新建查询，分割线，表，查询，函数，用户
3. 点击顶部菜单新建连接之后，弹出窗口，现选择数据库类型再编辑数据库连接信息
