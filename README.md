### 项目目标

跨平台数据库管理软件

#### 前提

1. 使用RUST的iced库构建页面
2. 项目需要兼容亮色和暗色主题


#### 主体框架

1. 构建顶，左，右的布局，顶部是图标菜单，左侧是连接管理，右侧是主要内容区域
2. 添加顶部菜单，顶部菜单为：新建连接，新建查询，分割线，表，查询，函数，用户
3. 点击顶部菜单新建连接之后，弹出子窗口，现选择数据库类型再编辑数据库连接信息
4. 为每个数据库类型都创建独立的连接编辑页面
5. 资源统一放在了assets目录下

#### 侧边栏定义
1. 侧边栏显示用户添加的数据库连接列表
2. 双击连接项后，显示连接中信息弹窗，连接成功后加载主窗口内容（参考主窗口定义）
3. 连接项添加右键菜单，包含，查看，编辑，删除等功能


#### 主窗口定义

主窗口 由tab构成
根据不通的类型，会有多种tab页面进行展示

tab分为固定tab和可删除tab

根据顶部表，查询，函数，用户四个菜单进行渲染的是固定tab
就是第一个tab页面，这个页面是不可以被删除的，根据点击不同的菜单渲染不同的内容
