### 项目目标

跨平台数据库管理软件

#### 前提

1. 使用RUST的iced库构建页面
2. 项目需要兼容亮色和暗色主题


#### 主体框架

1. 构建顶，左，右的布局，顶部是图标菜单，左侧是连接管理，右侧是主要内容区域
2. 添加顶部菜单，顶部菜单为：新建连接，新建查询，分割线，表，查询，函数，用户
3. 点击顶部菜单新建连接之后，弹出子窗口，现选择数据库类型再编辑数据库连接信息
4. 为每个数据库类型都创建独立的连接编辑页面
5. 资源统一放在了assets目录下

#### 侧边栏定义
1. 侧边栏显示用户添加的数据库连接列表
2. 双击连接项后，显示连接中信息弹窗，连接成功后加载主窗口内容（参考主窗口定义）
3. 连接项添加右键菜单，包含，查看，编辑，删除等功能


#### 主窗口定义
主窗口内容为tab布局，分为一个固定布局和其他可删除布局
根据顶部表，查询，函数，用户四个菜单进行展示，默认显示表列表
点击对应菜单，在固定布局内展示对应布局内容

对于表，查询，函数，用户四个菜单显示的具体内容要根据数据库类型详细定义。需要为每个数据库类型都创建对应页面组件

##### mysql

表页面展示当前数据库的所有表信息
查询页面展示用户保存的所有查询SQL
函数页面展示用户保存的所有函数
用户页面展示对应数据库的所有用户

其他数据库待定，先完成mysql数据库内容


#### 主窗口定义详细

对于表页面展示
分为顶部菜单和内容区域
顶部菜单
1. 打开表
2. 设计表
3. 新建表
4. 删除表
5. 导入向导
6. 导出向导
7. 切换展示模式

内容区域展示表列表，两种模式
1. 列表展示，展示表详细信息，一行一个，超出后纵向滚动
2. 网格展示，按照行展示，超出后添加新的列，超出后横向滚动


列表展示表时，展示图标+表名+详细信息
网格展示表时，只展示图标+表名
以上两种情况，都不要外边框，都需要添加选中状态
